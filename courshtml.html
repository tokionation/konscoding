Introduction au HTML :

HTML (HyperText Markup Language) est le langage standard utilisé pour structurer et présenter le contenu sur le Web. Il se compose d'éléments appelés balises, qui décrivent la structure et le formatage du contenu. Un document HTML est composé de balises HTML, de texte et de balises imbriquées les unes dans les autres.

Structure de base d'un document HTML :

    <!DOCTYPE html>
    <html>
    <head>
      <title>Titre de la page</title>
    </head>
    <body>
      <h1>Titre principal</h1>
      <p>Paragraphe de texte.</p>
    </body>
    </html>

Balises de texte :

Voici quelques balises couramment utilisées pour le texte :

  • <h1> à <h6> : balises de titre de différents niveaux.
  • <p> : balise pour les paragraphes.
  • <strong> ou <b> : balises pour mettre en évidence un texte en gras.
  • <em> ou <i> : balises pour mettre en évidence un texte en italique.
  • <u> : balise pour souligner un texte.
  • <br> : balise pour insérer un saut de ligne.
  • <hr> : balise pour insérer une ligne horizontale.
